## Project Overview
This repository contains the ongoing development infrastructure for the CSE 325 Group Project. For the initial planning phase, the following application concepts are being brainstormed and evaluated under the .NET ecosystem:

1. **Personal Garden Log** (Web App / .NET Blazor) - A digital diary and reminder system for home gardening enthusiasts.
2. **IELTS Vocabulary & Study Tracker** (Web App / .NET Blazor) - A personalized dashboard and flashcard tool for exam preparation.
3. **Space & Science Exploration Hub** (Mobile App / .NET MAUI) - A mobile platform aggregating space news and open-source science APIs.

## Development Setup
* **Framework:** .NET 8.0 / Blazor
